НАЗВАНИЕ:
	ConsoleGame - временное.

ОБЩЕЕ ОПИСАНИЕ:
	Игра для консоли Windows (командной строки). Пока все сводится к тому, что персонаж игрока бродит по комнатам мрачного подземелья, совершенно не освещаемого. Хотя у персонажа в руках есть масляная лампа, с ограниченным запасом масла.

ЦЕЛЬ ИГРЫ:
	- Найти дверь, ведущую к выходу из подземелья и выбраться из него.

ПРОИГРЫШ:
	- Если у персонажа закончится масло в горящей лампе, которая погаснет и он останется в кромешной темноте.

ПРАВИЛА:
	- В каждом помещении, называемом модулем карты, персонажу необходимо находить масло для лампы и ключи, чтобы иметь возможность перейти в следующий модуль.
	- Горящая лампа в руках игрока всегда освещает лишь небольшой радиус вокруг игрока, а все, что за пределами света, полностью скрыто (даже то, что персонаж уже прошел).
	- Персонаж может двигаться пошагово по горизонтали/вертикали на одну клетку. С каждой пройденной клеткой запас масла в лампе уменьшается. До того, как масло закончится необходимо найти новую банку с маслом.
	- Стены подземелья - это решетки, через которые проникает свет. Такое решение принимается для упрощения, чтобы не просчитывать загораживание света стенами. Когда будет сделан, описанный в этом документе минимум, очень возможно, что добавятся "глухие" стены, не пропускающие свет.
	- Найденные ключи могут открывать лишь одну любую дверь после чего исчезают. Дверь остается всегда открытой. Открытые двери позволяют перейти в следующий модуль. Возможно, будут двери и внутри модуля.
	- Для повышения интереса, возможно, напишу записки, рассказывающие историю подземелья и дающие подсказки. Внутри каждого модуля может появиться не более одной записки, но записки может и не быть.

МОДУЛИ КАРТЫ:
	- Вручную будет создан набор из нескольких модулей.
	- Каждый новый модуль игра будет выбирать случайно из набора. Если понадобится, то реализую возможность задавать шанс появления каждого модуля (вес).
	- В стенах модуля будет несколько специальных тайлов, на месте которых могут появиться или не появиться двери. Двери будут появляться случайно при создании нового модуля. Количество дверей можно будет ограничивать минимум и максимумом.
	- На месте любого специального тайла может появиться особенная дверь - выход из подземелья. Шанс появления такой двери изначально будет нулевым и повышаться при генерации каждого нового модуля.
	- Каждый модуль из набора при генерации нового модуля может быть повернут на угол, кратный 90 градусам.
	- Банки с маслом также будут создаваться случайно внутри модуля и могут появляться в любом месте, где нет стен/дверей. Минимум и максимум банок также будет задаваться для каждого модуля.

РЕДАКТОРЫ:
	- Дополнительная сложность при создании игры состоит  в том, что я хочу сделать максимально удобные (в рамках консоли) инструменты и редакторы для создания пикселей, тайлов, модулей карты.
